# Стиль кода

## Python

### Переменные

Используется `lower_snake_case` для именования переменных. Название переменной не содержит сокращений за исключением установленных ниже.

#### Сокращения для именования переменных виджетов `tkinter`

В начале переменной, содержащей виджет пишется его тип, а затем название через `_`.

- `w` — window
- `l` — label
- `b` — button
- `e` — entry
- `rb` — radio button
- `f` — frame
- `lf` — label frame
- `m` — menu

### Функции

Используется `CamelCase` для именования функций. Сокращения не допускаются. Название функции должно иметь в себе глагол, обозначающий то, что делает функция.

### Пример

```python
def QuitProgram():
    w_main.quit()
```
